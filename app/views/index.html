Stop Screens

  • Company has no current directors
  • Company type has no directors
  • Company is dissolved
  • Limited Partnership - Has partners, not directors. Can only file on paper.
  • Overseas company - Can only file on paper
  • Unregistered company - Can only file on paper
  • Limited Liability Partnership (LLP) - Has members, not directors. Can only file on paper or via WebFiling.
  • UK Societas - Has members, not directors. Can only file on paper.
  • Scottish qualifying partnership - Has partners,not directors. Can only file on paper or via Upload a doc.
  • Use paper form 288b if director was removed before 1st October 2009
