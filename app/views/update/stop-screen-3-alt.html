What happens if you update this director's home address

This address is currently recorded as the 'same as the correspondence address'. This means that any updates you make to this director's correspondence address also apply to their home address.

However, if you update the home address, it will no longer be recorded as the 'same'. This means that if you make updates to the correspondence address in the future, they will no longer apply to the home address as well.

If you need to update the home address, but want to keep the addresses recorded as the 'same', update the director's correspondence address instead.
